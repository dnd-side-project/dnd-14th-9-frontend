# 디자인 토큰 (Design Tokens)

## 토큰 구조

```
tokens/
├── default-token.json    # Figma Tokens Studio 출력
└── ...

src/styles/tokens/
├── theme.css             # Style Dictionary 변환 결과 (Tailwind v4)
└── variables.css         # CSS 변수
```

---

## 사용 방법

### Tailwind 클래스로 사용

```tsx
// 색상
<div className="bg-background-default text-text-primary" />
<div className="bg-green-500 text-common-white" />

// 간격
<div className="p-md gap-sm" />

// 형태
<div className="rounded-lg border-sm border-border-gray-default" />

// 타이포그래피
<h1 className="font-pretendard font-bold" />
```

### CSS 변수로 사용

```css
.custom-element {
  background-color: var(--color-green-500);
  padding: var(--spacing-md);
  border-radius: var(--radius-lg);
}
```

## 토큰 업데이트 워크플로우

1. **Figma**에서 디자인 토큰 수정
2. **Tokens Studio**로 JSON 내보내기
3. `tokens/` 폴더에 JSON 파일 업데이트
4. `pnpm tokens` 실행 (Style Dictionary 변환)
5. `src/styles/tokens/theme.css` 자동 생성

---

## 개발 명령어

```bash
# 토큰 변환 (1회)
pnpm tokens

# 토큰 변환 (watch 모드, dev 서버와 함께 실행됨)
pnpm tokens:watch

# 개발 서버 (토큰 watch 포함)
pnpm dev
```

## 네이밍 규칙

### Primitive Colors

- `{color}-{scale}`: `gray-500`, `green-300`, `red-700`

### Semantic Colors

- `{category}-{variant}`: `background-default`, `surface-primary-subtle`
- `{category}-{state}-{variant}`: `text-status-negative-default`

### Spacing & Shape

- `{size}`: `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`
