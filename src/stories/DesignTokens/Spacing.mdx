## 사용 가이드라인

### 컴포넌트 내부 여백

- 버튼 패딩: `p-sm` ~ `p-md`
- 카드 패딩: `p-lg` ~ `p-xl`
- 모달 패딩: `p-xl` ~ `p-2xl`

### 컴포넌트 간 간격

- 인라인 요소: `gap-xs` ~ `gap-sm`
- 폼 필드: `gap-md`
- 섹션 간: `gap-xl` ~ `gap-2xl`

### 레이아웃 간격

- 페이지 여백: `p-lg` ~ `p-2xl`
- 섹션 간격: `gap-2xl` ~ `gap-3xl`
